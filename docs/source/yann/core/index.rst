:mod:`core` - Core module contains the fundamentals operations of the toolbox 
=============================================================================

The module :mod:`core` contains the work horse of the ``yann`` toolbox. These are very
fundamental operations that are not needed to be known by or used by a user, but other modules
in the toolbox will use these constantly.
